# Inteligencia Artificial Proyecto IIB

## Descripción

Este proyecto utiliza un clasificador de sentimientos basado en el algoritmo **Naive Bayes** para analizar mensajes de texto. El modelo clasifica los mensajes como **positivos** o **negativos**. El proyecto incluye una interfaz gráfica de usuario (GUI) construida con **Tkinter** que permite a los usuarios interactuar con el modelo, clasificando mensajes en tiempo real.
